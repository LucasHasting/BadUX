In web design, user experience is a very important topic to keep in mind as you design your pages and interfaces. I will show the difference between a poor user experience and a decent user experience.

Bad UX

Example of a proffessor memory game that has a poor user experience by not giving the user any feedback to having a correct guess.

Better UX

A corrected version of the proffessor memory game that adds a fade effect to give the user a better experience.
